Ricocheting Robots
Fall 2019
Language: C++
Data Structures Used: Trees, Vectors, Linked Lists, Structs
Object-Oriented: Yes

Description

Ricocheting Robots is a program designed to use a tree data structure to solve a maze. It was modeled after the game "Ricochet Robot" (see this link: Ricochet Robots Wikipedia) Ricocheting Robots is designed to read a 10x10 gameboard and, given the position of the robot, obstacles and destination, find the shortest route to the destination. Alternatively, the user could also play an interactive version of the game.

Implementation

  • Read the gameboard from a file. Translate it to a 10x10 array of structs. Each struct represents one tile, and tells the robot whether it can move up, down, left or right at that tile.
  • Build a tree of solutions by traversing the gameboard
  • Find the shortest solution, and display

Sorting Algorithms
Fall 2019
Language: C++
Data Structures: Vectors, Structs Object-Oriented: No

Description

Implemented Insertion Sort, QuickSort and Radix Sort Algorithms for a set of positive numbers read from a file. This project resulted in the following observations:
  • Insertion Sort is best for mostly-sorted lists, and slow on unsorted ones
  • QuickSort works well for any lists
  • Radix Sort is very quick for lists with numbers close in range. For example, it would be useful to sort test scores of students, because those numbers are all within the range 0 - 100
